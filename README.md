## Installation
1. Clone the repository.
```git clone https://github.com/BlossomTheme/Cava.git```

2. Copy the ```config``` file to ```~/.config/cava```.
```cp ./config ~/.config/cava```

3. Run cava.
```
cava

or 

cava -p ~/.config/cava/config (if necessary)

```

4. Enjoy!
